Juplite v3.0 skeleton

Files:
- index.html   — main landing with embedded Jupiter Plugin
- swap.html    — direct swap page with deep link support (?pair=SOL-USDC)
- mobile.html  — mobile / PWA view, with Phantom / Solflare deep links
- manifest.json — PWA manifest
- styles.css   — dark, minimal UI theme
- script.js    — common JS (footer year helper)

Important:
- Icons referenced in HTML/manifest are not included here:
    favicon.ico
    icon-192.png
    icon-512.png
    ui-logo-64.png
    og.jpg

Replace them with your own assets in production.
